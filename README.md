# fileupload. 
This repo is about how to create a fileupload server using express,mongoose,multer.

    
## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd 
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Tech Stack

**Client:** Ejs

**Server:** Node, Express
