# fileupload. 
This repo is about how to create a fileupload server using express,multer.

    
## Run Locally

Clone the project

```bash
  git clone https://github.com/B-chandru/fileupload.git
```

Go to the project directory

```bash
  cd fileupload
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Tech Stack

**Client:** Ejs

**Server:** Node, Express, Multer.
